
========================================================================
1. OVERVIEW
========================================================================
This folder contains the complete solution for Task 2 (Male Target vs Female Interference in Reverberant Environment).
The model is a Convolutional Recurrent Network (CRN) that has been:
1. Pre-trained on Anechoic data.
2. Fine-tuned specifically for this Reverb task.

========================================================================
2. FOLDER CONTENTS
========================================================================
- process_task2.m           : MATLAB script to verify audio & package the final .mat file.
- target_signal.wav         : The clean target audio (Ground Truth).
- processed_signal.wav      : The AI-cleaned audio output (Generated by our model).
- Task2_Reverberant_5dB.mat : The FINAL submission file generated by the .m script.

[Model/ Subfolder]
- CRN_Reverb_finetuned.pth  : The trained model weights (The "Brain").
- inference_CRN.py          : Python script to run the model on new files.
- crn_finetune.py           : Source code used for training.

========================================================================
3. HOW TO RUN THE FILES
========================================================================

--- STEP 1: GENERATE AUDIO (Python) ---
(If you need to re-generate the output from scratch)
1. Navigate to the 'Model' folder.
2. Install requirements if needed: 
   pip install torch torchaudio numpy torchmetrics[audio]
3. Run the inference script:
   python inference_CRN.py
4. This will output a file named 'processed_signal.wav'. Move this file to the main folder if it isn't there already.

--- STEP 2: PACKAGE SUBMISSION (MATLAB) ---
(If you need to re-create the .mat file)
1. Open MATLAB and navigate to this 'Task2_Reverberant' folder.
2. Ensure these two files exist in the folder:
   - 'target_signal.wav'
   - 'processed_signal.wav'
3. Run the command:
   process_task2
4. If successful, it will create 'Task2_Reverberant_5dB.mat'.

========================================================================
4. TROUBLESHOOTING
========================================================================
- "File not found": Ensure the wav files are renamed EXACTLY to 'target_signal.wav' and 'processed_signal.wav'.
- "Sampling Rate Error": The script enforces 16kHz. Do not use 44.1kHz files.
